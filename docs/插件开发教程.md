## 1. 项目初始化

### 1.1 创建模板项目

使用 Halo 官方插件模板创建项目：

```bash
# 克隆模板
git clone https://github.com/halo-sigs/plugin-starter my-plugin
cd my-plugin
```

生成的项目结构：

```
my-plugin/
├── build.gradle              # Gradle 构建配置
├── settings.gradle
├── gradle.properties         # 插件版本
├── src/
│   ├── main/
│   │   ├── java/             # Java 后端代码
│   │   └── resources/
│   │       └── plugin.yaml   # 插件元数据
│   └── test/
└── ui/                       # 前端 Vue 项目
    ├── package.json
    ├── rsbuild.config.ts     # Rsbuild 配置
    └── src/
        ├── index.ts          # 插件入口（路由注册）
        └── views/            # Vue 组件
```

### 1.2 配置插件元数据

修改 `src/main/resources/plugin.yaml`：

```yaml
apiVersion: plugin.halo.run/v1alpha1
kind: Plugin
metadata:
  name: my-plugin           # 唯一标识，命名空间隔离
spec:
  enabled: true
  requires: ">=2.22.0"      # 最低 Halo 版本
  author:
    name: your-name
  displayName: "错题本"
  description: "记录、管理、复习错题"
```

### 1.3 配置 Java 工具链

```groovy
// build.gradle - 适配你本地的 JDK 版本
java {
    // toolchain { languageVersion = JavaLanguageVersion.of(21) } // 注释掉以使用系统 JDK
}
tasks.withType(JavaCompile).configureEach {
    options.encoding = "UTF-8"
    options.release = 21
}
```

### 1.4 安装前端依赖

```bash
cd ui
pnpm install    # 需要 Node.js 18+ / pnpm
```

## 2. 后端开发

### 2.1 数据模型 —— 自定义资源 (Extension)

在 Halo 中，所有数据都通过「自定义资源」管理。每个 Extension 需要：

- 继承 `AbstractExtension`
- 使用 `@GVK` 注解声明 Group/Version/Kind
- 定义一个内部 `Spec` 类存放业务字段

创建 `src/main/java/.../model/MistakeEntry.java`：

```java
@Data
@EqualsAndHashCode(callSuper = true)
@ToString(callSuper = true)
@GVK(group = "my-plugin.examole.djh",
     version = "v1alpha1",
     kind = "MistakeEntry",
     plural = "mistake-entries",
     singular = "mistake-entry")
public class MistakeEntry extends AbstractExtension {

    private MistakeEntrySpec spec;

    @Data
    public static class MistakeEntrySpec {
        private String subject;        // 科目
        private String question;       // 题目
        private String wrongAnswer;    // 错误答案
        private String correctAnswer;  // 正确答案
        private String analysis;       // 错题分析
        private String status;         // unmastered/reviewing/mastered
        private String difficulty;     // easy/medium/hard
        private String source;         // exam/homework/quiz/other
        private List<String> tags;     // 自定义标签
        private String knowledgePoint; // 知识点
        private String remark;         // 备注
    }
}
```

**关键注解说明**：

| 注解                                   | 说明                                        |
| ------------------------------------ | ----------------------------------------- |
| `@GVK`                               | 声明资源类型，group + version + kind 唯一标识        |
| `@Data`                              | Lombok，自动生成 getter/setter/toString/equals |
| `@EqualsAndHashCode(callSuper=true)` | 继承父类字段参与比较                                |

### 2.2 Scheme 注册 —— 必需步骤

Halo 不会自动发现插件中的自定义资源，需要手动注册 Scheme：

```java
@Component
public class MistakeEntrySchemeRegister {

    private final SchemeManager schemeManager;

    public MistakeEntrySchemeRegister(SchemeManager schemeManager) {
        this.schemeManager = schemeManager;
    }

    @EventListener(PluginStartedEvent.class)
    void onPluginStarted() {
        schemeManager.register(MistakeEntry.class);
    }
}
```

> ⚠️ 不加这一步会报错：`Scheme not found for my-plugin.examole.djh/v1alpha1/MistakeEntry`

### 2.3 业务逻辑——Service 层

创建 `src/main/java/.../service/MistakeEntryService.java`：

```java
@Slf4j
@Service
public class MistakeEntryService {

    private final ReactiveExtensionClient client;

    public MistakeEntryService(ReactiveExtensionClient client) {
        this.client = client;
    }

    // 分页查询（内存筛选，适用于数据量不大的场景）
    public Mono<ListResult<MistakeEntry>> listMistakeEntries(
            int page, int size, String subject, String status) {
        return listAll().collectList().map(all -> {
            List<MistakeEntry> filtered = all.stream()
                .filter(e -> subject == null ||
                    subject.equals(e.getSpec().getSubject()))
                .filter(e -> status == null ||
                    status.equals(e.getSpec().getStatus()))
                .collect(Collectors.toList());
            int from = (page - 1) * size;
            int to = Math.min(from + size, filtered.size());
            return new ListResult<>(page, size,
                (long) filtered.size(),
                from < filtered.size()
                    ? filtered.subList(from, to)
                    : List.of());
        });
    }

    // 获取所有
    public Flux<MistakeEntry> listAll() {
        return client.list(MistakeEntry.class, null, null);
    }

    // 创建（设置默认值）
    public Mono<MistakeEntry> createMistakeEntry(MistakeEntry entry) {
        if (entry.getSpec() == null)
            entry.setSpec(new MistakeEntry.MistakeEntrySpec());
        if (entry.getSpec().getStatus() == null)
            entry.getSpec().setStatus("unmastered");
        return client.create(entry);
    }

    // 更新
    public Mono<MistakeEntry> updateMistakeEntry(
            String name, MistakeEntry updated) {
        return client.get(MistakeEntry.class, name)
            .flatMap(existing -> {
                existing.setSpec(updated.getSpec());
                return client.update(existing);
            });
    }

    // 删除
    public Mono<Void> deleteMistakeEntry(String name) {
        return client.get(MistakeEntry.class, name)
            .flatMap(client::delete).then();
    }

    // 更新掌握状态
    public Mono<MistakeEntry> updateStatus(String name, String status) {
        return client.get(MistakeEntry.class, name)
            .flatMap(entry -> {
                entry.getSpec().setStatus(status);
                return client.update(entry);
            });
    }

    // 每日统计（365 天热力图数据）
    public Mono<List<DailyStats>> getDailyStats() {
        return listAll().collectList().map(entries -> {
            Map<LocalDate, int[]> dateMap = new LinkedHashMap<>();
            LocalDate today = LocalDate.now();
            LocalDate yearAgo = today.minusDays(365);
            for (MistakeEntry e : entries) {
                Instant created =
                    e.getMetadata().getCreationTimestamp();
                if (created == null) continue;
                LocalDate date = created.atZone(
                    ZoneId.systemDefault()).toLocalDate();
                int[] c = dateMap.computeIfAbsent(
                    date, k -> new int[2]);
                c[0]++;  // 新增数
                if ("mastered".equals(
                        e.getSpec().getStatus())) c[1]++;
            }
            List<DailyStats> result = new ArrayList<>();
            for (LocalDate d = yearAgo; !d.isAfter(today);
                 d = d.plusDays(1)) {
                int[] c = dateMap.getOrDefault(d, new int[2]);
                result.add(new DailyStats(
                    d.toString(), c[0], c[1]));
            }
            return result;
        });
    }

    public record DailyStats(
        String date, int added, int mastered) {}
}
```

### 2.4 REST API——Router 层

创建 `src/main/java/.../router/MistakeEntryRouter.java`：

```java
@Slf4j
@Configuration
public class MistakeEntryRouter {

    private final MistakeEntryService mistakeEntryService;

    @Bean
    RouterFunction<ServerResponse> mistakeEntryRoutes() {
        return RouterFunctions.route()
            .GET("/api/plugins/my-plugin/mistakes/stats/daily",
                this::getDailyStats)
            .GET("/api/plugins/my-plugin/mistakes",
                this::listMistakeEntries)
            .GET("/api/plugins/my-plugin/mistakes/{name}",
                this::getMistakeEntry)
            .POST("/api/plugins/my-plugin/mistakes",
                this::createMistakeEntry)
            .PUT("/api/plugins/my-plugin/mistakes/{name}",
                this::updateMistakeEntry)
            .DELETE("/api/plugins/my-plugin/mistakes/{name}",
                this::deleteMistakeEntry)
            .PATCH(
                "/api/plugins/my-plugin/mistakes/{name}/status",
                this::updateStatus)
            .build();
    }

    Mono<ServerResponse> listMistakeEntries(ServerRequest req) {
        int page = req.queryParam("page")
            .map(Integer::parseInt).orElse(1);
        int size = req.queryParam("size")
            .map(Integer::parseInt).orElse(20);
        String subject = req.queryParam("subject").orElse(null);
        String status = req.queryParam("status").orElse(null);
        return mistakeEntryService
            .listMistakeEntries(page, size, subject, status)
            .flatMap(r -> ServerResponse.ok().bodyValue(r));
    }
    // ... 其余 endpoint 实现
}
```

**API 一览**：

| 方法     | 路径                        | 说明                                 |
| ------ | ------------------------- | ---------------------------------- |
| GET    | `/mistakes`               | 分页列表 (page, size, subject, status) |
| GET    | `/mistakes/{name}`        | 单个详情                               |
| POST   | `/mistakes`               | 创建                                 |
| PUT    | `/mistakes/{name}`        | 更新                                 |
| DELETE | `/mistakes/{name}`        | 删除                                 |
| PATCH  | `/mistakes/{name}/status` | 更新状态                               |
| GET    | `/mistakes/stats/daily`   | 每日统计                               |

### 3.2 API 调用方式

Halo 插件前端使用 `axiosInstance` 调用后端 API：

```typescript
import { axiosInstance } from '@halo-dev/api-client'
import { Toast } from '@halo-dev/components'

// 获取列表
const resp = await axiosInstance.get(
  '/api/plugins/my-plugin/mistakes',
  { params: { page: 1, size: 20, subject: '数学' } }
)

// 创建
await axiosInstance.post('/api/plugins/my-plugin/mistakes', {
  spec: { subject: '数学', question: '...' }
})
Toast.success('添加成功')

// 删除
await axiosInstance.delete(`/api/plugins/my-plugin/mistakes/${name}`)
```

### 3.3 列表页核心逻辑

```typescript
import { ref, watch } from 'vue'
import { axiosInstance } from '@halo-dev/api-client'

const entries = ref([])
const page = ref(1)
const filterSubject = ref<string>()

// 监听筛选条件变化自动刷新
watch([filterSubject], () => {
  page.value = 1
  fetchEntries()
})

async function fetchEntries() {
  const resp = await axiosInstance.get(
    '/api/plugins/my-plugin/mistakes',
    { params: { page: page.value, subject: filterSubject.value } }
  )
  entries.value = resp.data.items
}
```

## 6. 常见问题

### Q: Scheme not found 错误

**现象**：`Scheme not found for my-plugin.examole.djh/v1alpha1/MistakeEntry`

**原因**：自定义 Extension 未注册到 SchemeManager。

**解决**：添加 SchemeRegister 组件（见 2.2 节）。

### Q: 前端路由不生效

**现象**：点击菜单后空白 / 404

**排查**：

1. 检查 `index.ts` 中 `parentName: 'Root'` 是否正确
2. 确保静态资源路径 `/stats` 在 `/:name` 之前注册
3. 检查前端组件导入路径

### Q: JDK 版本不匹配

**现象**：`Cannot find Java installation matching {languageVersion=21}`

**解决**：注释 `build.gradle` 中的 toolchain 块，使用系统 JDK。

### Q: Docker 镜像拉取失败

**现象**：`./gradlew haloServer` 卡住

**解决**：手动下载 Halo JAR，`java -jar` 直接启动。

### Q: Halo 登录的 RSA 加密

Halo 2.x 登录密码使用 RSA 公钥加密后提交：

```bash
# 1. 从 /login 页面获取 publicKey
# 2. 用 openssl 加密密码
echo -n "password" | openssl rsautl -encrypt -pubin \
  -inkey pubkey.pem -pkcs | base64
# 3. 提交加密后的密码到 /login
```
