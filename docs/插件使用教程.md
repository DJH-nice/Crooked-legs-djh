# 错题本插件 - 使用手册

## 简介

错题本插件帮助你记录、管理和复习错题。支持按科目分类、掌握状态追踪、难度标注、标签系统，以及 365 天学习热力图。

---

## 一、安装与启用

### 安装

1. 下载 `plugin-my-plugin-1.0.0-SNAPSHOT.jar`
2. 进入 Halo 后台 → **插件** → **上传**
3. 选择 JAR 文件上传

### 启用

上传后，在插件列表找到 **「错题本」**，点击右边开关启用。

启用后，左侧菜单会出现 **学习工具 → 错题本**。

### 卸载

Halo 后台 → 插件 → 找到错题本 → 点击 **卸载**。

---

## 二、功能概览

| 页面   | 路径             | 功能                        |
| ---- | -------------- | ------------------------- |
| 错题列表 | 左侧菜单「错题本」      | 查看所有错题，按科目/状态筛选，快速切换掌握状态  |
| 错题详情 | 点击任意题目         | 查看完整信息，对比正确答案，查看分析        |
| 学习统计 | 列表页顶部「📊 学习统计」 | 365 天贡献热力图，总记录/已掌握/连续打卡数据 |
| 添加错题 | 列表页「➕ 添加错题」    | 录入新错题                     |
| 编辑错题 | 列表/详情页「编辑」     | 修改已有错题                    |

## 三、添加错题

点击错题列表页右上角 **「➕ 添加错题」**，填写以下信息：

### 必填字段

- **科目**：数学、语文、英语、物理、化学、生物、历史、地理、政治、其他
- **题目**：题目原文（支持多行文本）

### 可选字段

- **错误答案 / 正确答案**：你的答案和正确答案对比
- **难度**：简单 / 中等 / 困难
- **来源**：考试 / 作业 / 测验 / 其他
- **知识点**：如"二次函数最值"
- **标签**：输入文字后按回车添加，可添加多个
- **错题分析**：分析错误原因和解题思路
- **备注**：其他补充信息

填写完成后点击 **「添加」** 保存。

## 四、管理和复习

### 筛选

在列表页使用筛选栏：

- **科目下拉**：只看某一科目的错题
- **状态下拉**：只看未掌握 / 复习中 / 已掌握的错题

选择筛选项后列表会自动刷新。

### 快速切换状态

点击列表中题目旁的 **状态标签**（未掌握 / 复习中 / 已掌握），即可快速切换：

```
未掌握 → 复习中 → 已掌握 → 未掌握（循环）
```

绿色 = 已掌握，橙色 = 复习中，红色 = 未掌握。

### 查看详情

点击列表中任意题目卡片，进入详情页。

详情页展示：

- 科目、状态、难度、来源标签
- 题目原文
- **错误答案 vs 正确答案** 对比
- 错题分析
- 知识点、标签、备注

### 编辑 / 删除

- **编辑**：列表页或详情页点击「编辑」按钮
- **删除**：点击「删除」按钮并确认

### 分页

当错题超过 10 条时，底部会出现分页按钮。

## 五、学习统计

点击列表页顶部 **「📊 学习统计」** 进入统计页。

### 概览卡片

- **总记录数**：已录入的全部错题数量
- **已掌握**：状态为「已掌握」的错题数
- **连续打卡天数**：最近连续有记录的天数

### 热力图

365 天的学习记录热力图，每个小格子代表一天：

- ⬜ 无色 = 当天无记录
- 🟩 绿色越深 = 当天记录越多

鼠标悬浮在格子上可查看具体日期和记录数。

底部有图例说明（少 → 多），顶部有月份标签。

## FAQ

### Q: 错题数据存在哪里？

A: 数据存在 Halo 的 H2 数据库中，随 Halo 备份一同保存。

### Q: 能导入/导出吗？

A: 当前版本暂不支持，可通过 API 自行开发（`GET /api/plugins/my-plugin/mistakes` 导出 JSON）。

### Q: 如何分享错题？

A: 可以将题目内容复制到 Halo 文章中分享。
